# Session Log — M-SRV、HDS-SRV 与裂缝体积 V_f 的关系

Date: 2026-05-13 17:22 CST

## 本次记录内容

本次讨论围绕压裂/储层评价中的两个 SRV 指标及其与我们计算的裂缝体积 \(V_f\) 的关系。

## 关键结论

1. **M-SRV 与 HDS-SRV 单位相同**
   - 表头中的 `×10^4 m³` 表示表内数值需乘以 \(10^4\) 立方米。
   - 二者差异主要在**定义和计算口径**，不是单位差异。

2. **M-SRV 的理解**
   - M-SRV 通常可理解为 **Microseismic SRV**，即基于微地震事件点云/事件包络估算的改造体积。
   - 它更接近“微地震活动覆盖范围”或“压裂影响范围”。
   - 该体积中可能包含无效事件、不连通裂缝、已闭合裂缝或对产能贡献有限的破裂区域。

3. **HDS-SRV 的理解**
   - HDS-SRV 通常可理解为在微地震结果基础上，进一步结合破裂机制、裂缝连通性、裂缝有效性等约束得到的 SRV。
   - 它更偏向“有效改造体积”或“有效连通缝网体积”。
   - 因此，HDS-SRV 一般比 M-SRV 更接近有效产能相关的改造范围。

4. **与我们计算的 \(V_f\) 的关系**
   - 我们计算的 \(V_f\) 表征的是**裂缝储液体积/缝内容积**，不是储层空间改造体积。
   - 其物理意义更接近有效水力裂缝内部可容纳流体的体积。
   - 已对齐的公式口径为：

     \[
     V_f = \frac{\pi I_F L H_w^2 P_{net}}{E'}
     \]

     其中 \(L\) 为裂缝半长，\(H_w\) 为缝高，\(P_{net}\) 为净压力，\(E'\) 为平面应变模量，\(I_F\) 为宽度剖面积分项。

5. **推荐表述**

   可以说：

   > 相比 M-SRV，HDS-SRV 对无效微地震事件和非连通破裂体积进行了约束，因此更接近有效改造体积。由于本文反演的 \(V_f\) 表征的是有效水力裂缝的缝内储液体积，其变化主要受裂缝长度、高度、净压力和裂缝柔度控制，因此 \(V_f\) 与 HDS-SRV 的相关性应强于与 M-SRV 的相关性。但二者物理含义不同，HDS-SRV 表征储层空间改造范围，\(V_f\) 表征裂缝内部储存体积，不能直接等同。

6. **一句话版**

   > HDS-SRV 和我们算的 \(V_f\) 更相关，但 \(V_f\) 是“缝内容积”，HDS-SRV 是“有效改造空间体积”；二者相关，不等价。

## 注意事项 / 未验证点

- 本记录基于当前会话中的讨论和用户提供的公式语境整理。
- 这里未重新检索或核验具体文献中 `HDS-SRV` 的严格缩写定义；如用于论文或正式报告，建议回到原始数据表、方法说明或对应文献确认 HDS 的完整定义与计算流程。
- 不应将 HDS-SRV 直接写成等于 \(V_f\)。更稳妥的说法是：**HDS-SRV 与 \(V_f\) 的相关性预期强于 M-SRV 与 \(V_f\) 的相关性**。

## 后续建议

- 若后续有 M-SRV、HDS-SRV 和 \(V_f\) 的样本数据，可计算 Pearson/Spearman 相关系数，并画散点图比较：
  - \(V_f\) vs. M-SRV
  - \(V_f\) vs. HDS-SRV
- 若 HDS-SRV 明显与 \(V_f\) 更相关，可在报告中作为“有效改造体积比微地震包络体积更能反映裂缝储液能力”的支持证据。
