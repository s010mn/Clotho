# Clotho

**Clotho: 基于停泵数据的压裂缝网参数评价方法研究**

Clotho 是一个硕士毕业论文研究项目，用于整理和实现基于停泵数据/停泵压力响应的压裂缝网参数评价方法。

## 研究主题

- 停泵压力响应与 DFIT/G-function 等解释方法
- 压裂裂缝闭合、储层渗流与压力衰减特征
- 压裂缝网参数评价、解释与可能的反演方法
- 文献资料、研究笔记、数据处理脚本与论文写作材料

## 项目结构

```text
.
├── AGENTS.md              # 项目研究指南与任务约束

├── experiments/           # 后续实验代码与运行记录（待添加）
├── outputs/               # 后续分析报告与图表输出（待添加）
└── papers/                # 后续论文草稿与写作材料（待添加）
```

## 命名说明

Clotho 是希腊神话中命运三女神之一，意为“纺线者”。本项目借用“线”的意象：从停泵数据这条线索出发，解析地下压裂缝网这一隐蔽结构。
